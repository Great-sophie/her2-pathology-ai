# HER2 Pathology AI Analysis

This repository contains a deep learning pipeline for analyzing HER2
immunohistochemistry (IHC) pathology images.

## Motivation
HER2 is an important biomarker in breast cancer. This project explores
CNN-based methods for HER2 image classification and interpretation.
